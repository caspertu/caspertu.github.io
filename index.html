可以看到express主要包含三个核心概念：路由、中间件、模板引擎。

  • 中间件：在 express 应用中，一切皆中间件。各种应用逻辑，如 cookie 解析、会话处理、日志记录、权限校验等，都是通过中间件来完成的。
  • 路由：用户发送的http请求，该定位到哪个资源，就是路由说了算。
  • 模版引擎：负责视图动态渲染。

如何进入DEBUG模式

设置DEBUG的值为express:router，只查看路由部分的日志。

CNAME文件丢失问题

github 中设置Custom domain后，会在 github 上生成CNAME文件，文件内容是指定的跳转域名。

但是每次执行hexo d后，github 上的CNAME文件会被自动删除，导致自定义域名不可用。

解决办法很简单，把CNAME文件放在 hexo的source目录下就可以了。

亲爱的唐棠姐姐，你好

很高兴最近爸爸有很多时间和你在一起，养成一些好习惯，例如早起。我们早上7点起床，晨跑，打羽毛球，溜滑板，赏花，运气好的话还能在小区里遇到你的小伙伴。早起不仅能让你8点前按时到达学校，也能更好的利用每天的时间，何乐而不为。

我们还做了很多有趣的事情，经过几次在线课后，你越来越喜欢编程。我们一起制作了会自动寻找路线的机器人小车，使用模块化编程让电脑版闪烁，显示你和妹妹的英文名字，还能按照你的指令发出悦耳的声音。在乐高的 Wedo 编程课上，你的表现非常出色。

每次写成长日记，都觉得时间过的好快，不是吗？一转眼，我的小宝贝就要幼儿园毕业成为小学生了。爸爸也觉得最近的你越来越懂事，无论是在阅读、数学和英语学习上，还是你喜欢的围棋、手工制作等，甚至你原来不太擅长的跳绳，通过几天的练习，你从连续跳2、3次进步到22次。你成长快得让爸爸有些意外。

希望爸爸能一直陪你成长，克服困难和阻碍，实现你的梦想。
